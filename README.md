# SYSC Assignment 3

## Compilation

To compile the project, run:

```bash
source build.sh
```
## Execution

To run the EP scheduler, use the following command:

```bash
./bin/interrupts_EP <input_file.txt>
```

To run the EP scheduler, use the following command:
```bash
./bin/interrupts_RR <input_file.txt>
```

To run the EP scheduler, use the following command:
```bash
./bin/interrupts_EP_RR <input_file.txt>
```
